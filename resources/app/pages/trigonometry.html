📖 Important Formulas

sin θ

= Perpendicular / Hypotenuse

cos θ

= Base / Hypotenuse

tan θ

= Perpendicular / Base

cot θ

= Base / Perpendicular

sec θ

= Hypotenuse / Base

cosec θ

= Hypotenuse / Perpendicular

Identity

sin²θ + cos²θ = 1

✍ Example

Find sin θ

Perpendicular = 3
Hypotenuse = 5

sin θ = 3 / 5 = 0.6

Find tan θ

Perpendicular = 8
Base = 6

tan θ = 8 / 6 = 4 / 3
