📖 Important Formulas

Differential Calculus

d(xⁿ)/dx = n·xⁿ⁻¹

d(c)/dx = 0

d(x)/dx = 1

d(sin x)/dx = cos x

d(cos x)/dx = −sin x

d(tan x)/dx = sec²x

Integral Calculus

∫xⁿ dx = xⁿ⁺¹/(n+1) + C

∫dx = x + C

∫sin x dx = −cos x + C

∫cos x dx = sin x + C

∫eˣ dx = eˣ + C

∫1/x dx = ln|x| + C
